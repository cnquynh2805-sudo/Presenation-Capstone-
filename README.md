# Bloom Capstone Deck

Bloom is a capstone presentation deck for a friendly habit tracker concept. The deck introduces the problem, product direction, technical approach, user journey, live demo mockup, future roadmap, and team workflow.

## Overview

Bloom focuses on helping users build consistent habits through a simple tracking experience, motivational progress feedback, NFC quick check-ins, and a personalized companion mascot.

This repository contains a static web presentation built with plain HTML, CSS, and JavaScript.

## Features

- Slide-based capstone presentation
- Keyboard navigation with left and right arrow keys
- Previous and next slide controls
- Visual progress indicator and slide counter
- Responsive 16:9 presentation layout
- Habit tracker phone mockup
- Mascot GIF assets and story images

## Tech Stack

- HTML
- CSS
- JavaScript
- Google Fonts

## How To Run

Open `index.html` directly in a browser.

For a local server, run one of these commands from the project folder:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Navigation

- Click the arrow buttons at the bottom right to move between slides.
- Press the right arrow key to go to the next slide.
- Press the left arrow key to go to the previous slide.

Bloom combines core habit tracking features with a more encouraging and low-friction experience:
